### 2.1 Discount Lots (discountlots.com)

| Özellik | Değerlendirme |
|---------|--------------|
| **Kuruluş** | 2015, Florida |
| **İş modeli** | Kendi envanterini taksitli satış |
| **Envanter** | 200+ aktif listing |
| **Fiyat aralığı** | $2,000 - $30,000 |
| **Taksit seçenekleri** | 12-60 ay |
| **Hedef pazar** | Sadece ABD |
| **Web sitesi** | Orta kalite, eski tasarım |
| **Ödeme sistemi** | Manuel ACH / PayPal |
| **AI kullanımı** | Yok |
| **Tahmini aylık gelir** | $50,000-100,000 MRR |

**Güçlü yanları:**
- Yılların deneyimi ve müşteri tabanı
- Geniş envanter

- Eski teknoloji, mobil uyumsuz tasarım
- Sadece ABD pazarına odaklı
- Manuel ödeme takibi (ölçeklenmez)
- Müşteri deneyimi zayıf
- AI veya otomasyon yok

### 2.2 LandWatch (landwatch.com)

| Özellik | Değerlendirme |
|---------|--------------|
| **Kuruluş** | 2004, Texas |
| **İş modeli** | İlan platformu (marketplace) |
| **Envanter** | 500,000+ listing (3rd party) |
| **Hedef pazar** | ABD |
| **Gelir modeli** | Reklam + premium listing ücreti |
| **Web sitesi** | İyi kalite |

**Güçlü yanları:**

**Zayıf yanları:**

**TerraLot için fırsat:** LandWatch'tan piyasa fiyatlarını çekerek comparable sales analizi yapılabilir.

## 5. PAZAR PAYI PROJEKSİYONU

### Mevcut pazar dağılımı (tahmini):

| Oyuncu Tipi | Pazar Payı | Yıllık İşlem |
|-------------|-----------|-------------|
| Bireysel satıcılar | %70 | $21 milyar |
| Orta ölçekli firmalar | %20 | $6 milyar |
| Büyük platformlar | %10 | $3 milyar |

### TerraLot hedefi (Yıl 1):

```
Toplam pazar:        $30 milyar
Erişilebilir pazar:  $3 milyar (taksitli segment)
Hedef pazar payı:    %0.01
Hedef gelir:         $300,000
```

Bu muhafazakar bir hedef. Teknoloji avantajı ve global erişim ile %0.05'e ulaşmak 3 yıl içinde mümkündür ($1.5M).
